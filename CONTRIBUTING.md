# Contributing

## Development

1. Use Windows PowerShell 5.1 or PowerShell 7 on Windows.
2. Install Node.js 18 or newer and run `npm ci`.
3. Keep local API keys in environment variables or the ignored `Start-CodexWorkflow.config.json` file.
4. Run `npm run check` and `npm run release:check` before opening a pull request.

Keep changes focused and include tests for shared parsing, state, or process behavior.
